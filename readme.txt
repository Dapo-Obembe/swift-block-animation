== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/swift-block-animation` directory, or install via the WordPress Plugins screen.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Edit any page or post with the block editor (Gutenberg).
4. Select a block ‚Üí open the **Entrance Animation** panel in the sidebar ‚Üí choose your animation.
5. Save and preview your page to see animations in action.

== Frequently Asked Questions ==

= Where do I find the animation settings? =
Select a block in the editor, then open the **Entrance Animation** panel in the right sidebar.  
If you don‚Äôt see the sidebar, click the **gear ‚öôÔ∏è icon** in the editor toolbar.

= Do animations work on mobile? =
Yes, animations are mobile-friendly. If a user has "reduced motion" enabled on their device for accessibility, animations will be disabled automatically.

= Can I animate multiple elements in sequence? =
Yes! Use the **Stagger Animation** option to animate galleries, columns, or grouped blocks one after another.
